Om de scraper te runnen moet je python en de packages die je in versies.txt vind geinstaleerd hebben.
de packages kun je handmatig downloaden of door de commands in versie.txt uit te voeren in command prompt

Handleiding:
1. Open command prompt in de map waar de scraper zig bevind of
voer het command cd locatievanscraper uit (bijvoorbeeld: cd C:\Users\wvanz\Documents\GitHub\football-league-manager\Prototypes)
2. Activeer de scraper door het command python prototypeScraper.py uit te voeren
